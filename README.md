# 🌍 WanderLust - Travel Destination Web App

**WanderLust** is a full-stack travel web application where users can explore, share, and review travel destinations around the world. It features user authentication, image uploads, dynamic content rendering, and a responsive interface built using the **MVC architecture**.

🔗 **GitHub Repository:** [github.com/ashmitsingh13/WanderLust](https://github.com/ashmitsingh13/WanderLust)

---

## ✨ Features

- 🧭 View, create, edit, and delete travel destinations
- 🔐 Secure user authentication & authorization (Login/Signup)
- 🖼️ Image uploads using **Multer** and **Cloudinary**
- 📝 Add, edit, and delete reviews with star ratings
- 📱 Fully responsive and mobile-friendly layout
- ⚠️ Flash messages and custom error handling
- ✅ Form validations for better user experience

---

## 🛠️ Tech Stack

### Backend:
- **Node.js**
- **Express.js**
- **MongoDB** (with **Mongoose**)
- **Passport.js** (for authentication)

### Frontend:
- **EJS** (Embedded JavaScript templates)
- **HTML5**
- **CSS3**
- **Bootstrap 5**

### Other Tools:
- **Multer** (for handling image uploads)
- **Cloudinary** (image hosting)
- **Express-Validator** (form validation)
- **Connect-Flash** (flash messaging)
- **Method-Override**
- **Dotenv**
